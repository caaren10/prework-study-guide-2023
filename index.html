HTML

  • HTML is made up of elements which provide context and basic framework for the webpage
  • There are different types of elements: text elements, image elements, header elements, and list elements
  • The head element contains information about the webpage. These contents are not visible when the user eneter the webpage. Metadata helps the browser render the page correctly. This element is a container for metadata. This is content is used for Search Engine Optimization (SEO).
  • The body element represents the visible content shown to the user

Git

  • git status: checks what branch we are currently on
  • git checkout -b [branch-name]: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch
  • git commit -m "[description]": this code commits the file and adds a message with what the commit contains
  • git pull: this command receives a branch's modifications in the local environment
  • git push origin: this command pushes the changes made locally to the remote GitHub branch. the origin code indicates the sorice of the pull will be in the GitHub repo.
